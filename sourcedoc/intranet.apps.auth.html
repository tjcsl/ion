Authenticate using Kerberos.

This is the default authentication backend.

authenticate(username=None, password=None)[source]¶

Authenticate a username-password pair.

Creates a new user if one is not already in the database.

Args:
username
The username of the User to authenticate.
password
The password of the User to authenticate.
Returns:
User

NOTE: None is returned when the user account does not exist. However, if the account exists but does not exist in LDAP, which is the case for former and future students who do not have Intranet access, a dummy user is returned that has the flag is_active=False. (The is_active property in the User class returns False when the username starts with “INVALID_USER”.)
