Note: Due to the nature of the .doc and .docx file format, the printing output may not be what you expected. Hence, we do not explicitly support printing those file formats, although they will still print. If possible, print PDF documents for accurate results.

The following file formats are supported:
  • .pdf: Adobe PDF
  • .ps: Adobe PostScript
  • .odt: OpenDocument Text Document
  • .txt: Plain Text

Do NOT attempt to print image files or non-supported file formats.
The following restrictions apply:
  • Print jobs can have a maximum of 10 pages.
  • If the printer doesn't appear to be working, do NOT keep attempting to print. Instead, contact the Student Systems Administrators with the Ion feedback form.
  • Please be mindful of other printing users.
  • Abuse of this feature will result in loss of printing privileges.
