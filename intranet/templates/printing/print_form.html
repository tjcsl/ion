The following file formats are supported:
  • .docx: Microsoft Word XML Document
  • .doc: Microsoft Word Document
  • .pdf: Adobe PDF
  • .ps: Adobe PostScript
  • .odt: OpenDocument Text Document

The following restrictions apply:
  • Print jobs can have a maximum of 15 pages.
  • You should check with a teacher before printing to a non-public location.
  • If the printer doesn't appear to be working, don't keep attempting to print.
  • Abuse of this feature will result in loss of printing privileges.
